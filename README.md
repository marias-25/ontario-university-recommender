# ontario-university-recommender
A web-based, values-aligned university recommendation system for Ontario students. Built as course project for GNG5301 (Professional Skills and Responsibilities) at the University of Ottawa.

Website: https://ontario-university-recommender-en.streamlit.app/

The app matches students with the best-fit universities from Ontario's top 10 institutions based on their academic profile and personal priorities — including employment outcomes, cost of living, sustainability, diversity, campus feel, and co-op availability. Powered by a transparent weighted scoring model using open data from CUDO and institutional sources.

Built with Python and Streamlit.
